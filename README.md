# Customer Churn Analysis

## Project Overview
This project analyzes customer behavior and identifies drivers of churn to help businesses improve retention. Using Python, SQL, and Power BI, we processed large datasets, built predictive models, and delivered actionable insights.

## Key Features
- **Data Processing:** Cleaned, transformed, and explored 50K+ customer records using Python and SQL.
- **Predictive Modeling:** Built logistic regression models to quantify the impact of features on churn.
- **Dashboarding:** Created Power BI dashboards for cohort analysis, retention trends, and funnel visualization.
- **Automation:** Integrated Databricks pipelines to refresh dashboards automatically, improving reporting efficiency.

## Skills & Tools
- **Languages:** Python, SQL
- **Analytics & BI Tools:** Power BI, Databricks
- **Techniques:** Feature engineering, exploratory data analysis, logistic regression, cohort analysis, automated reporting

## Outcome
- Identified key factors driving customer churn and recommended targeted retention strategies.
- Delivered self-updating dashboards for business stakeholders to track retention KPIs in real time.
